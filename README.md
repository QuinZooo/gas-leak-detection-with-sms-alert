A safety monitoring dashboard that detects gas leaks in real-time using IoT sensors.

Project Description

GasGuard is a comprehensive safety monitoring application that monitors air quality. It uses an MQ-2 sensor to detect gas leaks and instantly updates a Flutter dashboard while triggering an SMS alert via a Cloud API.

 Getting Started

```bash
# Clone the repository
git clone https://github.com/QuinZooo/gas-leak-detection-with-sms-alert.git

# Navigate to project directory
cd gasguard_app

# Install dependencies
flutter pub get

# Run the app
flutter run
```

instructions

1. Configure your IoT device (MQ-2 sensor)
2. Set up cloud services
3. Add your Promotexter API credentials to the app configuration
4. Update dashboard settings as needed
