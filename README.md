# GasGuard - Gas Leak Detection with SMS Alert

A safety monitoring dashboard that detects gas leaks in real-time using IoT sensors.

## Project Description

GasGuard is a comprehensive safety monitoring application that monitors air quality. It uses an MQ-2 sensor to detect gas leaks and instantly updates a Flutter dashboard while triggering an SMS alert via a Cloud API.

## Technologies Used

- **Mobile**: Flutter, Dart
- **IoT**: MQ-2 Sensor
- **Backend**: Cloud Computing
- **Notifications**: Promotexter SMS API

## Features

- Real-time status indicators
- Cloud logging and analytics
- Remote mute functionality
- Instant SMS alert notifications
- Responsive dashboard UI

## Installation

### Prerequisites
- Flutter SDK installed
- Dart SDK
- IoT device setup (MQ-2 sensor)
- Cloud services configured
- Promotexter API credentials

### Getting Started

```bash
# Clone the repository
git clone https://github.com/QuinZooo/gas-leak-detection-with-sms-alert.git

# Navigate to project directory
cd gasguard_app

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## Setup Instructions

1. Configure your IoT device (MQ-2 sensor)
2. Set up cloud services
3. Add your Promotexter API credentials to the app configuration
4. Update dashboard settings as needed
