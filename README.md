Project Title: GAS LEAK DETECTION W/ SMS ALERT
Description: A safety monitoring dashboard that monitors air quality. It uses an MQ-2 sensor to detect leaks and instantly updates a Flutter dashboard while triggering an SMS via a Cloud API.
Technologies Used: Flutter, Dart, IoT, Cloud Computing, Promotexter.
Features: Status Indicators, Cloud Logging, Remote Mute
Installation Instructions: Instructions on how to clone and run the app locally (for others or future you).
Setup: Instructions for setting up IoT devices and cloud services.
